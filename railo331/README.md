DESCRIPTION
===========

Sets up Railo 3.3.1.000. Currently supports Railo Express only.

REQUIREMENTS
============

ATTRIBUTES
==========

For Railo
--------------

* `node['railo']['install_path']` - Railo installation path (default: "/opt/railo")
* `node['railo']['webroot']` - Railo webroot (default: "/vagrant/webroot")
